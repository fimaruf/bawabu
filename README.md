OAuth 2 protocol server 
=======================
Currently implements the following grants:
* authorization code
* user credentials
* work-in progress

### install dependencies
Best served with [RVM](https://rvm.org) :-)

``
$ bundle install
``

### Tests
``
$ rspec
``
